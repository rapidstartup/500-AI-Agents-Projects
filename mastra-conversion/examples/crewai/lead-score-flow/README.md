# Lead Score Flow (MASTRA)

A workflow agent that scores and qualifies leads based on multiple criteria.

## Features

- Lead data analysis
- Scoring algorithm
- Qualification threshold
- Priority ranking

## Setup

```bash
npm install
```

## Environment

Copy `.env.example` to `.env` and add your API keys:

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
