# Meeting Assistant (MASTRA)

A workflow-based agent that helps prepare for and manage meetings.

## Features

- Agenda generation
- Participant research
- Summary creation
- Action item extraction

## Setup

```bash
npm install
```

## Environment

Copy `.env.example` to `.env` and add your API keys:

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
