# Game Builder Crew (MASTRA)

A multi-agent supervisor system for building games with design, code, and testing agents.

## Features

- Game designer agent
- Code generator agent
- QA tester agent
- Supervisor coordination

## Setup

```bash
npm install
```

## Environment

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
