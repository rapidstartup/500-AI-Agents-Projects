# Marketing Strategy Generator (MASTRA)

A workflow agent that generates comprehensive marketing strategies.

## Features

- Market analysis
- Target audience identification
- Channel strategy
- Content planning
- Budget allocation

## Setup

```bash
npm install
```

## Environment

Copy `.env.example` to `.env`:

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
