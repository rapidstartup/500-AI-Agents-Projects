# Surprise Trip Planner (MASTRA)

A multi-agent supervisor that plans surprise trips with researcher, planner, and coordinator agents.

## Features

- Destination research
- Itinerary planning
- Booking coordination
- Surprise element planning

## Setup

```bash
npm install
```

## Environment

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
