# MASTRA AI Health Assistant

**Type:** Standalone Agent  
**Conversion Pattern:** Single Agent + Tools (Template A)  
**Complexity:** Medium

## Overview

MASTRA conversion of standalone AI Health Assistant agent.

## What It Does

- BMI calculation from height/weight
- General health information and wellness tips
- Health metric explanations
- Lifestyle improvement suggestions

## Running

```bash
npm install
cp .env.example .env
# Add OPENAI_API_KEY
npm run demo
```

## Note

This agent provides general health information only. Always consult healthcare professionals for medical advice.
