# MASTRA Virtual AI Tutor

**Type:** Standalone Agent

**Complexity:** Medium

## Overview

MASTRA conversion of Virtual AI Tutor agent.

## Running

npm install
cp .env.example .env
# Add OPENAI_API_KEY
