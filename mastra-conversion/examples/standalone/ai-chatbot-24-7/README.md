# 24/7 AI Chatbot (MASTRA)

A round-the-clock AI chatbot with conversation flow handling and escalation.

## Features

- 24/7 availability
- Intent recognition
- Conversation flows
- Escalation handling
- Context retention

## Setup

```bash
npm install
```

## Environment

```
OPENAI_API_KEY=your_openai_key
```

## Run

```bash
npm run dev
```
