# Automated Trading Bot (MASTRA)

An agent that executes automated trading strategies based on market analysis.

## Features

- Market data analysis
- Signal generation
- Trade execution
- Risk management
- Portfolio rebalancing

## Setup

```bash
npm install
```

## Environment

```
OPENAI_API_KEY=your_openai_key
ALPHA_VANTAGE_API_KEY=your_alpha_vantage_key
```

## Run

```bash
npm run dev
```
